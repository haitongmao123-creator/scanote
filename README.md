# scanote

一个智能 PDF/Word 文档识别与命名工具，支持本地模式和 AI 模式。

## 功能特点

- **支持 PDF 和 Word 文档**（.pdf / .doc / .docx）
- **AI 模式**：调用智谱 AI glm-4v-flash 模型，支持图片识别，完全免费
- **本地模式**：无需网络，使用 PDF.js 直接提取文本
- **自动命名**：识别文档标题，按规则自动重命名文件
- **自定义命名规则**：可配置前缀、标题、日期的组合与顺序
- **一键下载**：处理完成后直接下载重命名后的文件

## 使用方法

1. 打开页面，将 PDF 或 Word 文件拖入上传区域
2. 系统自动识别文档内容并生成文件名
3. 点击下载获取重命名后的文件

如需使用 AI 模式，请在设置中填写智谱 AI 的 API Key（免费申请：https://open.bigmodel.cn/）
